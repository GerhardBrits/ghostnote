# Ghostnote 👻

**WhatsApp is for chat. Ghostnote is for secrets.**

Send anything that self-destructs after one view. No accounts. No history. No screenshots.

Live demo: https://yourusername.github.io/ghostnote/

---

### How it works
1. Type your secret in Ghostnote
2. Get a one-time link: `ghost.link/a7x9k`
3. Send the link on WhatsApp, SMS, or email
4. Receiver opens it once — 30 second timer — then it's cryptographically shredded

Optional **Split Key**: send the link in one app, the 4-word code in another for true two-channel security.

### Features
- End-to-end encrypted in your browser (libsodium)
- Screenshot & screen-record blocked on mobile
- No signup, no phone number, no cloud storage
- Works as a PWA — install to home screen like a native app
- Burn-on-open, not burn-on-timer
